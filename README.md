# 🏁 RaceWithPros (Release)

**RaceWithPros** is a synchronization tool designed to connect your smart bike trainer to the profile of a professional cycling race in real-time. Experience the climbs and descents exactly as the pros do while you watch the broadcast.

## 🚀 Current Support
This alpha version is optimized for:
*   **Strade Bianche 2025**
*   **Strade Bianche 2026**

---

## 🛠 Features
*   **Real-Time Sync:** Uses OCR (Optical Character Recognition) to read the "KM to go" directly from a race broadcast on your screen.
*   **Smart Trainer Control:** Automatically adjusts your trainer's resistance based on the race's elevation profile (supports Bluetooth FTMS).
*   **Live Telemetry:** View your power, heart rate, and speed alongside the race leader's progress.
*   **Compact Mode:** An overlay mode that lets you keep the race broadcast front and center.

## 📥 How to Install & Run (Windows)

1.  **Download:** Go to the [**Releases**](https://github.com/GylleTanken/RaceWithPros-Releases/releases) page and download the latest `RaceWithPros_vX.X.X.X.zip`.
2.  **Extract:** Right-click the ZIP file and select **"Extract All..."**. 
    > ⚠️ **IMPORTANT:** You must keep the folder structure as in the extracted ZIP file. The app will not run if you move the `.exe` out of the folder.
3.  **Run:** Double-click `racewithpros_flutter.exe`.
4.  **Security Note:** Since this is an unsigned Alpha version, Windows SmartScreen may show a warning. Click *"More info"* and then *"Run anyway"*.

## 🧪 Alpha Testing Notes
*   **Hardware:** Requires a smart trainer that supports the Bluetooth FTMS protocol.
*   **OCR Setup:** For the best experience, use the "READ KM" tool in the app to select the area on your screen where the race's "KM to go" timer is displayed.
*   **Feedback:** This is an early alpha. Bugs are expected! Please report any issues or feedback via [Reddit] or the [GitHub Issues] tab on this repository.
